#  Chat App

A full-stack real-time chat application built with:

-  React + Vite (Frontend)
 - Zustand (State Management)
- Socket.io (Real-time Communication)
- Express + Node.js (Backend)
- MongoDB Atlas (Database)
- Render (Deployment)
